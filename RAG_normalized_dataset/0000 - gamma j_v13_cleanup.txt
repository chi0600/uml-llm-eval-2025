**Title:** Remove Plug-ins – Removal Process

**Raw:**

Requirements:  
Remove Plug-ins

Goal:  
Remove a plug-in from the application.

Actors:  
Administrator  
Weborder System

Preconditions:  
Administrator must be able to access the weborder system via a web browser with HTTPS.

Triggers:  
Administrator clicks the button or link to “Remove Plug-ins”

Basic Scenario:  
1. Administrator clicks the button or link to initiate plug-in deletion process.  
2. System prompts the Administrator to select the desired plug-in module.  
3. System removes the plug-in and validates changes.

Postconditions:  
A plug-in is removed from the application.

---

**Clean:**  
**Actors:** Administrator, Weborder System  
**Preconditions:** Administrator can access the Weborder System via HTTPS.  
**Triggers:** Administrator clicks the **Remove Plug-ins** button or link.  

**Flow:**  
1. Administrator → Weborder System: Click “Remove Plug-ins”  
2. Weborder System → Administrator: Prompt to select plug-in module  
3. Administrator → Weborder System: Select plug-in module  
4. Weborder System → Weborder System: Remove and validate plug-in  
5. Weborder System → Administrator: Confirm removal  

**Alternative Scenarios:**  
- None

**Postconditions:**  
Plug-in is removed and confirmation is displayed to the Administrator.
