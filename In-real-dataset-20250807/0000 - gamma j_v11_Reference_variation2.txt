Change User Properties

Alter properties such as passwords and privileges of the user.

- Administrator
- Weborder System

Administrator clicks the button or link to "Change User Properties"

1. Administrator clicks the button or link to initiate the change user properties process.
2. System prompts the Administrator to select a user by searching or viewing a list of users.
3. System displays the user information.
4. Administrator modifies the user properties.
5. System updates the user properties in the system.
6. System displays the account home page to the Administrator.

Postconditions:
An account has been altered within the system.
