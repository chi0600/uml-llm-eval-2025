Add User

Goal:
Register a new customer, sales person, or administrator account with the system.

- Weborder System

Administrator clicks the button or link to "Add Users"

2. System prompts the Administrator to fill out first name, last name, username, email address, password, and privileges.
3. System validates the new user information.
4. System creates a new account for the new user with desired privileges.
5. System displays the account home page to the Administrator.

Postconditions:
A new customer account is created within the system.
